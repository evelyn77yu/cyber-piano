# 🎹 ZenType Cyber-Piano Pro
> **Transform your PDF textbooks into a Cyberpunk Piano Concert.** > **将你的 PDF 课本变成一场赛博朋克钢琴演奏会。**

---

## 🌟 Overview / 项目简介

**ZenType Cyber-Piano** is an immersive study tool designed for international students to tackle dense academic English materials (like Deep Learning textbooks). By integrating a **Cyberpunk-style typing interface** with a **Generative Piano Engine**, it turns tedious reading into a rhythmic, decompressing experience.

**ZenType Cyber-Piano** 是一款为留学生量身定制的沉浸式复习工具。它将**赛博朋克风格的打字界面**与**生成式钢琴引擎**结合，把枯燥的专业课 PDF 阅读变成了一场充满节奏感、极其解压的打字练习。

---

## ⌨️ Core Features / 核心功能

* **Cyber-Piano Engine / 赛博钢琴引擎**: 
    - Every correct keystroke plays a note from Beethoven's *Für Elise*. 
    - 每敲对一个字母，都会奏响《献给爱丽丝》的一个音符。
* **Case-Insensitive Typing / 不区分大小写**: 
    - Smooth typing experience without worrying about the Shift key. 
    - 全程无需切换大小写，输入行云流水。
* **Wildcard Skip (Tab) / 万能跳过键**: 
    - Use `Tab` to instantly skip complex LaTeX formulas or messy symbols. 
    - 遇到复杂的数学公式或乱码？按下 `Tab` 瞬间跳过。
* **Two Practice Modes / 双练习模式**: 
    - **Sequence**: Follow the original book structure. 
    - **Shuffle**: Randomly extract paragraphs for high-intensity review.
    - **顺序模式**：按课本逻辑推进；**随机模式**：全书随机抽题，强化记忆。
* **Deep Outline Parsing / 深度目录识别**: 
    - Automatically extracts PDF bookmarks for quick chapter navigation. 
    - 自动提取 PDF 电子目录，支持点击章节名直接跳转。

## 🚀 How to Play / 如何使用

1.  **Open the link** in your Desktop browser (Chrome/Edge recommended).
2.  **Upload** your academic PDF textbook.
3.  **Switch to English Input Method** (Crucial for keyboard event capturing).
4.  **Start Typing!** Each correct character syncs with the piano melody.
5.  **Decompress & Review**: Focus on the text while enjoying the music.

1.  在电脑浏览器中**打开链接**（推荐 Chrome 或 Edge）。
2.  **上传**你的英文专业课 PDF 课本。
3.  **切换至英文输入法**（这是确保按键识别准确的关键）。
4.  **开始打字！** 每一个正确的按键都会同步触发钢琴音效。
5.  **解压与复习**：在享受旋律的同时，高效完成专业课复习。

## 🛠️ Technical Implementation / 技术实现

- **Frontend**: Vue 3 (CDN), Tailwind CSS.
- **PDF Engine**: PDF.js for client-side local parsing (Your files are never uploaded to any server).
- **Audio**: Web Audio API (Generative synthesis with Hall Reverb).
- **Automation**: Global Window Key Listener.

- **前端框架**: Vue 3 (CDN), Tailwind CSS。
- **解析引擎**: PDF.js 本地解析（你的文件**不会**被上传到服务器，完全私密）。
- **音频系统**: Web Audio API (带大厅混响的生成式音效)。
- **交互逻辑**: 全局窗口按键监听。
